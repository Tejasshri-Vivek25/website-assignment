# Resume Website

A professional, modern, and responsive resume website for Tejasshri Vivek, built with HTML and CSS.

## Features

- **Clean Design**: Minimalist aesthetic with a professional color palette.
- **Responsive Layout**: Adapts seamlessly to different screen sizes (desktop, tablet, mobile).
- **Smooth Scrolling**: Anchor links with smooth scrolling behavior.
- **Card-Based Layout**: Organized content using distinct cards for experience and education.
- **Typography**: Uses Google Fonts (`Outfit` and `Playfair Display`) for a premium look.
- **Interactive Elements**: Hover effects on cards and navigation links.

## Color Palette

- **Primary**: `#0b132b` (Dark Navy)
- **Accent**: `#3b82f6` (Bright Blue)
- **Background**: `#f8fafc` (Very Light Blue-Gray)
- **Cards**: `#ffffff` (White)

## Structure

The website consists of a single HTML page (`index.html`) and a styling sheet (`style.css`).

- **Navigation Bar**: Sticky header with links to different sections.
- **Hero Section**: Introduction with name and subtitle.
- **Main Content**:
    - **Experience**: Professional roles and responsibilities.
    - **Education**: Academic qualifications and achievements.
    - **Skills**: Technical and soft skills.
- **Footer**: Contact information and copyright.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge).

### Installation

1.  Clone the repository or download the source files.
2.  Open `index.html` in your web browser.

## Usage

Simply open the `index.html` file in any web browser to view the resume.

## Technologies Used

- **HTML5**: For semantic structure.
- **CSS3**: For styling and layout.
- **Google Fonts**: For typography.
